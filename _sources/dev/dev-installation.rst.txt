Installation
============

To install a development version of the ``PARC`` package:

.. code:: bash

  git clone https://github.com/ahill187/PARC.git
  cd PARC
  pip install -e ".[dev]"


You will also need to install ``pandoc``:

.. code:: bash

  wget https://github.com/jgm/pandoc/releases/download/3.3/pandoc-3.3-1-amd64.deb
  sudo dpkg -i pandoc-3.3-1-amd64.deb
